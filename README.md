# 🎨 Análise de Colorimetria Pessoal

Aplicação web simples para upload e processamento de imagens com o objetivo de realizar uma análise de colorimetria pessoal.

## 📋 Descrição

Este projeto permite que o usuário selecione uma foto do seu dispositivo e envie a imagem para processamento. A aplicação foi desenvolvida utilizando HTML, CSS, JavaScript e Bootstrap para a interface.

## 🚀 Funcionalidades

* Upload de imagens (`.jpg`, `.jpeg`, `.png`, entre outros formatos suportados pelo navegador).
* Interface responsiva utilizando Bootstrap 5.
* Estilização personalizada com CSS.
* Processamento da imagem através de JavaScript.
* Área dinâmica para exibição dos resultados da análise.

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* Bootstrap 5.0.2
* Google Fonts
